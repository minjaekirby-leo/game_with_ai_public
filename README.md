# 10살 어린이를 위한 게임 개발 학습 프로젝트

이 프로젝트는 다양한 웹 기술을 사용해서 간단한 게임들을 만들어보는 학습용 프로젝트입니다.

## 프로젝트 구조

### 1. p5js-games/ - p5.js 라이브러리 사용
- `bouncing-ball/` - 튀는 공 게임
- `simple-pong/` - 간단한 퐁 게임

### 2. canvas-games/ - HTML5 Canvas + JavaScript 사용  
- `snake-game/` - 뱀 게임
- `catch-game/` - 떨어지는 물체 잡기 게임

### 3. scratch-style/ - Snap! (Scratch for JavaScript) 스타일
- `block-breaker/` - 블록 깨기 게임

## 실행 방법

각 폴더의 `index.html` 파일을 브라우저에서 열면 게임을 플레이할 수 있습니다.

## 온라인 에디터에서 테스트하기

- **p5.js 게임들**: [p5.js Web Editor](https://editor.p5js.org/)에 코드 복사
- **Canvas 게임들**: [CodePen](https://codepen.io/)이나 [JSFiddle](https://jsfiddle.net/)에서 테스트
- **모든 게임들**: [Replit](https://replit.com/)에서 전체 프로젝트 실행

## 학습 순서 추천

1. p5js-games/bouncing-ball 부터 시작
2. 각 게임의 코드를 읽고 이해하기
3. 작은 변경사항 만들어보기 (색깔, 속도, 크기 등)
4. AI에게 질문하면서 새로운 기능 추가하기
