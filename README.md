# 🎮 서민재의 게임랜드

AI와 함께 서민재가 만들어가는 재밌는 게임들을 함께 즐겨봐요~
**총 111개의 게임이 준비되어 있습니다!** 🎮

## 🆕 새로운 기능

### 새로운 게임 쇼케이스 시스템
- **7일간 새 게임 특별 전시**: 새로 추가된 게임들을 7일간 별도 섹션에서 특별 전시
- **자동 카운트다운**: 쇼케이스 종료까지 남은 시간을 실시간으로 표시
- **자동 이동**: 7일 후 자동으로 메인 게임 목록으로 이동
- **관리자 도구**: 게임 추가/삭제/관리를 위한 전용 관리 페이지

### 최신 추가 게임 (80년대 클래식 아케이드 컬렉션)
- **🟡 팩맨 게임**: 클래식 아케이드 게임의 완벽한 재현
  - 화살표 키로 팩맨 조종
  - 유령 AI와 파워 펠릿 시스템
  - 레벨 시스템과 점수 기록

- **🚀 갤러그 게임**: 클래식 슈팅 아케이드 게임
  - 화살표 키로 우주선 조종
  - 적들의 급강하 공격 패턴
  - 스페이스바로 미사일 발사

- **🕹️ 아케이드 컬렉션**: 6개 게임이 하나로!
  - 스페이스 인베이더, 브레이크아웃, 스네이크
  - 아스테로이드, 프로거, 갤러그
  - 각 게임별 최고점수 기록

- **🪨 아스테로이드**: 우주선으로 소행성 파괴
  - 화살표 키로 우주선 조종 및 회전
  - 스페이스바로 레이저 발사
  - 소행성 분할 시스템과 레벨 진행

- **🐛 센티피드**: 마우스 조준 슈팅 게임
  - 마우스로 조준, 클릭으로 발사
  - 센티피드 분할 메커니즘
  - 버섯 장애물과 거미/벼룩 적

- **🚀 미사일 커맨드**: 도시 방어 게임
  - 마우스 클릭으로 요격 미사일 발사
  - 연쇄 폭발 시스템
  - 6개 도시 방어 미션

- **🦍 동키콩**: 클래식 플랫폼 액션
  - 화살표 키로 마리오 조종
  - 배럴 피하기와 사다리 타기
  - 해머 파워업과 점수 시스템

- **🔺 Q*bert (피라미드 블록 색칠하기)**: 퍼즐 액션 게임
  - 화살표 키로 큐브 점프
  - 모든 블록 색깔 바꾸기
  - 적 회피와 보너스 아이템

- **⛏️ 디그더그**: 땅파기 액션 게임
  - 화살표 키로 이동, 스페이스바로 펌프
  - 적 부풀리기와 바위 떨어뜨리기
  - 터널 파기 전략 게임

- **🌀 템페스트**: 벡터 그래픽 슈팅
  - 화살표 키로 원형 이동
  - 스페이스바로 발사
  - 터널 깊이감과 벡터 스타일

- **🏍️ 트론 (라이트 사이클)**: 전략 액션 게임
  - 화살표 키로 방향 전환
  - AI 상대와의 대결
  - 벽 충돌 회피 전략

## 📁 프로젝트 구조

### 1. p5js-games/ - p5.js 라이브러리 사용
- `bouncing-ball/` - 튀는 공 게임
- `simple-pong/` - 간단한 퐁 게임

### 2. canvas-games/ - HTML5 Canvas + JavaScript 사용  
- `snake-game/` - 뱀 게임
- `catch-game/` - 떨어지는 물체 잡기 게임

### 3. scratch-style/ - Snap! (Scratch for JavaScript) 스타일
- `block-breaker/` - 블록 깨기 게임

### 4. new-games/ - 새로운 게임들
- `pacman/` - 팩맨 게임 (NEW!)
- `galaga/` - 갤러그 게임 (NEW!)
- `arcade-collection/` - 아케이드 컬렉션 (NEW!)
- 기타 100개의 다양한 게임들

### 5. new-games-showcase/ - 새 게임 쇼케이스 시스템
- 7일간 새 게임 특별 전시 페이지
- 자동 카운트다운 및 관리 시스템

### 6. admin/ - 관리자 도구
- `game-manager.html` - 게임 추가/삭제/관리 도구

## 🎮 주요 기능

### 새로운 게임 쇼케이스
1. **자동 타이머**: 새 게임이 7일간 특별 섹션에 표시
2. **실시간 카운트다운**: 남은 시간을 일/시간/분/초로 표시
3. **자동 정리**: 만료된 게임은 자동으로 메인 목록으로 이동
4. **관리자 도구**: 웹 인터페이스로 쉬운 게임 관리

### 팩맨 게임 특징
- **클래식 게임플레이**: 원작의 재미를 그대로 재현
- **스마트 AI**: 4개의 유령이 각각 다른 패턴으로 움직임
- **파워 시스템**: 파워 펠릿으로 유령을 잠시 먹을 수 있음
- **레벨 시스템**: 모든 점을 먹으면 다음 레벨로 진행
- **생명 시스템**: 3번의 기회로 도전

## 🚀 사용 방법

### 일반 사용자
1. `index.html`을 브라우저에서 열기
2. "🆕 새로운 게임 쇼케이스" 섹션에서 최신 게임 확인
3. 원하는 게임 클릭하여 플레이

### 관리자 (게임 추가)
1. `admin/game-manager.html` 열기
2. 새 게임 정보 입력 (ID, 제목, 아이콘, 설명 등)
3. 쇼케이스 기간 설정 (3일/7일/14일/30일)
4. "게임 추가" 버튼 클릭

### 개발자 (프로그래밍 방식)
```javascript
// 브라우저 콘솔에서 새 게임 추가
addNewGame({
    id: 'my-new-game',
    title: '내 새 게임',
    icon: '🎮',
    description: '재미있는 새 게임입니다!',
    tech: 'HTML5',
    path: '../new-games/my-game/index.html',
    showcaseDays: 7
});
```
